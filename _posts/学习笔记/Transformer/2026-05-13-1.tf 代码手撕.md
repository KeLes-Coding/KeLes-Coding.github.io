# 1.tf 代码手撕

按照以下路线：
```
第 1 步：Token Embedding + Positional Encoding
第 2 步：Scaled Dot-Product Attention
第 3 步：Multi-Head Attention
第 4 步：Padding Mask / Causal Mask
第 5 步：Feed Forward Network
第 6 步：Encoder Layer
第 7 步：Decoder Layer
第 8 步：完整 Encoder-Decoder Transformer
第 9 步：裁剪成 Encoder-only
第 10 步：裁剪成 Decoder-only
```

## 第 1 步：Embedding + Positional Encoding

Transformer 接收的原始输入通常是 token id：
```python
input_ids = [
    [12, 35, 89, 7, 0, 0],
    [45, 11, 23, 99, 31, 0],
]
```
形状为：
```text
[B, L]
```
其中：
```text
B = batch_size
L = seq_len
```
但是模型不能直接处理 token id。我们需要先把每个 token id 映射成一个向量。

所以第一步是：
```text
[B, L] -> [B, L, D]
```
其中：
```text
D = d_model
```

### 1. Token Embedding

先看最简单的 embedding。

```python
import torch
import torch.nn as nn

vocab_size = 1000   # 词表大小
d_model = 512       # 每个 token 映射成 512 维向量

embedding = nn.Embedding(vocab_size. d_model)

input_ids = torch.tensor([
    [12, 35, 89, 7, 0, 0],
    [45, 11, 23, 99, 31, 0],
])

x = embeding(input_ids)

print(x.shape)
```
