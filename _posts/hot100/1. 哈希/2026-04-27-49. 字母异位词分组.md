---

## 思路

目标：找到 字母异位词 的列表。

这道题的一个主要的点是 **字母异位词**，即字母出现次数相同的单词，那么其实对于这些词，使用 `sort(key.begin(), key.end());` 对每个词进行排序，排序的结果就是`key`，如果同属于一个 **字母异位词** 的词语的结构都将是一样的，这时候只需要通过一个 `unordered_map` 对他们进行存储即可。

至此我们也可以推出 `unordered_map` 的存储格式： `[string, vector<string>]`。

---

## 题解

```cpp
class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> map;
        vector<vector<string>> ans;

        for (string s : strs) {
            string key = s;
            sort(key.begin(), key.end());
            map[key].push_back(s);
        }

        for (auto& [k, v] : map) {
            ans.push_back(v);
        }

        return ans;
    }
};
```

## tips——关于 `auto`

它的作用是：**让编译器自动推导变量类型**。

也就是说，你不需要手动写出变量的完整类型，编译器会根据右边的值自动判断类型。

对于我们的代码：
```cpp
auto& [k, v]
```
这里的 `&` 表示引用，意思是不要复制 `map` 里的元素，而是直接引用它。

在这题里，v 可能是一个 `vector<string>`，如果不用引用，可能会产生不必要的拷贝。
