# 1. hot100-128. 最长连续序列

[题目链接](https://leetcode.cn/problems/longest-consecutive-sequence/description/?envType=study-plan-v2&envId=top-100-liked)

给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

 

**示例 1：**

**输入：** nums = [100,4,200,1,3,2]
**输出：** 4
**解释：** 最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

**示例 2：**

**输入：** nums = [0,3,7,2,5,8,4,6,0,1]
**输出：** 9

**示例 3：**

**输入：** nums = [1,0,1,2]
**输出：** 3

 

**提示：**
- $0 <= nums.length <= 10^5$
- $-10^9 <= nums[i] <= 10^9$

---

## 思路

题目要求找到数组中最长的连续整数序列，且时间复杂度为 `O(n)` 。

那么可以直接排除使用复杂度 `O(nlogn)` 的排序方法。

考虑使用 `unordered_set` 来存储数组中的数字，之所以不用 `unordered_map`，是因为当前场景只注重于查找是否存在而不需要获取其其他信息（如下标），且查到单个元素的时间复杂度为 `O(1)`。

然后是本题的一个核心代码：`if (!s.count(i - 1))`。

因为要查找最长连续整数序列，那么可以确定的是 假设该序列的一个数字为`x`，那么`x-1`不可能存在，我们只需要从`x`开始进行这个序列的查找即可，而 `unordered_set` 的特性使其天然适合查找 `x-1` 以及 `x+1`。

总结一下：
- 如果 `x - 1` 存在，说明 `x` 不是序列起点，它一定会在前面的起点扩展时被统计到，因此可以跳过。
- 如果 `x + 1` 存在，说明 `x` 不是序列终点，此时赋值 `x = x + 1`，继续查找。
---

## 题解

```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        int ans = 0;

        unordered_set<int> s(nums.begin(), nums.end());
        for (int i : s) {
            if (!s.count(i - 1)) {
                int cur = i;
                int len = 1;

                while (s.count(cur + 1)) {
                    cur++;
                    len++;
                }

                ans = max(ans, len);
            }
        }

        return ans;
    }
};
```
