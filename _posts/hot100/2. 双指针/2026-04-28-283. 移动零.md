# 283. 移动零
给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

**示例 1:**

> **输入:** nums = `[0,1,0,3,12]`
> 
> **输出:** `[1,3,12,0,0]`

**示例 2:**

> **输入:** nums = `[0]`
> 
> **输出:** `[0]`
 

**提示**:
- $1 <= nums.length <= 10^4$
- $-2^{31} <= nums[i] <= 2^{31} - 1$

进阶：你能尽量减少完成的操作次数吗？

---

## 思路

本题要求将 `0` 移动到末尾，同时非零元素相对顺序不变。

考虑构建 快慢双指针：

- `slow`， 表示下一个非 0 元素的下标；
- `fast`， 用来从左到右遍历数组。

那么其实核心的语句为 `nums[fast] != 0`，即如果快指针不为 0，那么将 `nums[slow]` 以及 `nums[fast]` 做一个 `swap`，因为 `slow` 是不做判定的，其表示就只是单纯的位置信息，或者也可以理解为，`fast` 遍历到现在，不为零的数的数量；而 `fast` 做是否为 0 的判定，只要不为零，那么当前 `slow` 所在的位置就应该是这个非零的数应该在的位置，然后做一个 `slow++`。

---

## 题解

```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0;

        for (int fast = 0; fast < nums.size(); fast++) {
            if (nums[fast] == 0) {
                continue;
            } else {
                swap(nums[slow], nums[fast]);
                slow++;
            }
        }
    }
};
```
更加简洁的版本：
```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow = 0;

        for (int fast = 0; fast < nums.size(); fast++) {
            if (nums[fast] != 0) {
                swap(nums[slow], nums[fast]);
                slow++;
            }
        }
    }
};
```
