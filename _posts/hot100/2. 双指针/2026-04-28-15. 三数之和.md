# 1. hot100-15. 三数之和

[题目链接](https://leetcode.cn/problems/3sum/description/?envType=study-plan-v2&envId=top-100-liked)

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：** 答案中不可以包含重复的三元组。

**示例 1：**

> **输入：** nums = [-1,0,1,2,-1,-4]
> 
> **输出：** [[-1,-1,2],[-1,0,1]]
> 
> **解释：**
> 
> nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
> 
> nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
> 
> nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
> 
> 不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
> 
> 注意，输出的顺序和三元组的顺序并不重要。

**示例 2：**

> **输入：** nums = [0,1,1]
> 
> **输出：** []
> 
> **解释：** 唯一可能的三元组和不为 0 。

**示例 3：**

> **输入：** nums = [0,0,0]
> 
> **输出：** [[0,0,0]]
> 
> **解释：** 唯一可能的三元组和为 0 。

 

**提示：**
- $3 <= nums.length <= 3000$
- $-10^5 <= nums[i] <= 10^5$

---

## 思路

这道题的目标是找出 `a + b + c == 0` 的三元组，且一个限制是**不重复**。

暴力法做一个三重循环能做，但是肯定会超时。优化的方案是，在外层用 `i` 做一个喜欢，然后内部使用 `双指针` 做一个匹配，或者说找到 `nums[left]` 以及 `nums[right]`，使 `nums[left] + nums[right] == -nums[i]`，即 `nums[left] + nums[right] + nums[i] == 0`。

首先是将 `nums` 做一个排序。然后固定第一个数 `nums[i]`，在 `i` 的右侧做 `双指针`。

接下来我们解析一下 `双指针` 的一个推进思路：
- 基础的循环条件： `while (l < r)`
- 然后就是根据 `sum` 的值来做处理：
  - `sum < 0`： 和太小，左指针右移。
  - `sum > 0`： 和太大，右指针左移。
  - `sum == 0`： 入库，然后两个指针均移动，同时注意跳过重复元素。

这道题还有一个比较麻烦的点是要求结果不能重复，因此当前有两层去重：
- 枚举 `i` 时，如果 `nums[i] == nums[i - 1]`，跳过；
- 找到答案后，移动 `l` 和 `r`，跳过相同的数字。

---

## 题解

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> ans;
        sort(nums.begin(), nums.end());
        for (int i = 0; i < nums.size(); i++) {
            if (nums[0] > 0) break;
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int l = i + 1;
            int r = nums.size() - 1;
            while (l < r) {
                long long sum = nums[i] + nums[l] + nums[r];
                if (sum < 0) {
                    l++;
                } else if (sum > 0) {
                    r--;
                } else {
                    ans.push_back({nums[i], nums[l], nums[r]});
                    l++;
                    r--;
                    while (l < r && nums[l] == nums[l-1]) l++;
                    while (l < r && nums[r] == nums[r+1]) r--;
                }
            }
        }

        return ans;
    }
};
```
