# 1. hot100-73. 矩阵置零

[题目链接](https://leetcode.cn/problems/set-matrix-zeroes/description/?envType=study-plan-v2&envId=top-100-liked)

给定一个 `m x n` 的矩阵，如果一个元素为 **0 **，则将其所在行和列的所有元素都设为 **0** 。请使用 **原地** 算法**。**

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/08/17/mat1.jpg)

> **输入：** matrix = [[1,1,1],[1,0,1],[1,1,1]]
> 
> **输出：** [[1,0,1],[0,0,0],[1,0,1]]


**示例 2：**

![](https://assets.leetcode.com/uploads/2020/08/17/mat2.jpg)

> **输入：** matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
> 
> **输出：** [[0,0,0,0],[0,4,5,0],[0,3,1,0]] 

**提示：**
- `m == matrix.length`
- `n == matrix[0].length`
- $1 <= m, n <= 200$
- $-2^{31} <= matrix[i][j] <= 2^{31} - 1$

**进阶：**
- 一个直观的解决方案是使用  `O(mn)` 的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用 `O(m + n)` 的额外空间，但这仍然不是最好的解决方案。
- 你能想出一个仅使用常量空间的解决方案吗？

---

## 思路

本题的要求是：如果矩阵中某个位置为 `0`，就将它所在的整行和整列都置为 `0`。

需要注意的是，不能在第一次遍历时直接把行列置零。因为这样会产生新的 `0`，影响后续判断，导致原本不该被置零的位置也被错误处理。

因此可以分两步：

1. 第一次遍历矩阵，记录哪些行、哪些列原本存在 `0`；
2. 第二次遍历矩阵，如果当前位置所在的行或列被标记过，就将该位置置为 `0`。

使用两个数组：

- `row[i]` 表示第 `i` 行是否需要置零；
- `col[j]` 表示第 `j` 列是否需要置零。

---

## 题解

```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int m = matrix.size();
        int n = matrix[0].size();

        vector<int> row(m, 0);
        vector<int> col(n, 0);

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (matrix[i][j] == 0) {
                    row[i] = 1;
                    col[j] = 1;
                }
            }
        }

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (row[i] == 1 || col[j] == 1) {
                    matrix[i][j] = 0;
                }
            }
        }
    }
};
```

## 复杂度分析

时间复杂度：`O(mn)`，要遍历矩阵两次。

空间复杂度：`O(m + n)`。
