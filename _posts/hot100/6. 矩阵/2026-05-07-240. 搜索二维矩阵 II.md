# 1. hot100-240. 搜索二维矩阵 II

[题目链接](https://leetcode.cn/problems/search-a-2d-matrix-ii/description/?envType=study-plan-v2&envId=top-100-liked)

编写一个高效的算法来搜索 `m x n` 矩阵 `matrix` 中的一个目标值 `target` 。该矩阵具有以下特性：
- 每行的元素从左到右升序排列。
- 每列的元素从上到下升序排列。


示例 1：

![](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/11/25/searchgrid2.jpg)
```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 5
输出：true
```

示例 2：

![](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2020/11/25/searchgrid.jpg)
```
输入：matrix = [[1,4,7,11,15],[2,5,8,12,19],[3,6,9,16,22],[10,13,14,17,24],[18,21,23,26,30]], target = 20
输出：false
```

**提示：**
- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= n, m <= 300`
- $-10^9 <= matrix[i][j] <= 10^9$
- 每行的所有元素从左到右升序排列
- 每列的所有元素从上到下升序排列
- $-10^9 <= target <= 10^9$

---

## 思路

矩阵的每一行从左到右递增，每一列从上到下递增。

可以从右上角开始搜索。右上角元素有一个特点：

- 它是当前行的最大值；
- 它是当前列的最小值。

因此，设当前位置为 `matrix[i][j]`：

如果等于 `target`，直接返回 `true`；

如果大于 `target`，说明当前列下面的数只会更大，所以这一列可以排除，向左移动；

如果小于 `target`，说明当前行左边的数只会更小，所以这一行可以排除，向下移动。

不断缩小搜索范围，直到越界。

---

## 题解

```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();

        int i = 0;
        int j = n - 1;

        // 这个矩阵的主要特点是，右上角的数字为本行最大、本列最小。
        // 因此，只要 target 大于右上角，那么应该去下面的行查找
        // 如果 target 小于左上角，那么就从右往左遍历、比较过去
        while (i < m && j >= 0) {
            if (matrix[i][j] == target) {
                return true;
            } else if (matrix[i][j] > target) {
                j--;
            } else {
                i++;
            }
        }

        return false;
    }
};
```

## 复杂度分析

时间复杂度：`O(m + n)`，最多向左移动 n 次，向下移动 m 次。

空间复杂度：`O(1)`。
