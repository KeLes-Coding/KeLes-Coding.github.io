# 1. hot100-56. 合并区间

[题目链接](https://leetcode.cn/problems/merge-intervals/description/?envType=study-plan-v2&envId=top-100-liked)

以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]` 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。

 

**示例 1：**

> **输入：** intervals = [[1,3],[2,6],[8,10],[15,18]]
> 
> **输出：** [[1,6],[8,10],[15,18]]
> 
> **解释：** 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

**示例 2：**

> **输入：** intervals = [[1,4],[4,5]]
> 
> **输出：** [[1,5]]
> 
> **解释：** 区间 [1,4] 和 [4,5] 可被视为重叠区间。

示例 3：

> **输入：** intervals = [[4,7],[1,4]]
> 
> **输出：** [[1,7]]
> 
> **解释：** 区间 [1,4] 和 [4,7] 可被视为重叠区间。

 

**提示：**
- $1 <= intervals.length <= 10^4$
- $intervals[i].length == 2$
- $0 <= start_i <= endi <= 10^4$

---

## 思路

本题的核心是：先按照区间左端点排序，这样可能发生重叠的区间一定会相邻。

排序后，维护一个当前正在合并的区间 `[start, end]`。

遍历每个区间 `[curStart, curEnd]`：

- 如果 `curStart <= end` （这个 `=` 很重要），说明当前区间和维护中的区间有重叠，可以合并：
  ```cpp
  end = max(end, curEnd);
  ```
- 如果 `curStart > end`，说明两个区间不重叠，当前维护的区间已经结束，将 `[start, end]` 加入答案，然后开启新的区间：
    ```cpp
    ans.push_back({start, end});
    start = curStart;
    end = curEnd;
    ```

然后记得：遍历结束后，还需要把最后一个维护中的区间加入答案。

---

## 题解

```cpp
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        vector<vector<int>> ans;

        if (intervals.empty()) return ans;

        sort(intervals.begin(), intervals.end());

        int start = intervals[0][0];
        int end = intervals[0][1];

        for (int i = 0; i < intervals.size(); i++) {
            int curStart = intervals[i][0];
            int curEnd = intervals[i][1];

            if (end >= curStart) {
                end = max(end, curEnd);
            } else {
                ans.push_back({start, end});
                start = curStart;
                end = curEnd;
            }
        }

        ans.push_back({start, end});

        return ans;
    }
};
```

## 复杂度分析

时间复杂度：`O(nlogn)`，主要来自排序。

空间复杂度：`O(logn)` 或 `O(n)`，取决于排序实现和返回结果空间。
