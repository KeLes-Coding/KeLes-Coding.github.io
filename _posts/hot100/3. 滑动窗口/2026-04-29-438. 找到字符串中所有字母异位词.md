# 1. hot100-438. 找到字符串中所有字母异位词

[题目链接](https://leetcode.cn/problems/find-all-anagrams-in-a-string/?envType=study-plan-v2&envId=top-100-liked)

给定两个字符串 `s` 和 `p`，找到 `s` **中所有 `p`** 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

 

**示例 1:**

> **输入:** s = "cbaebabacd", p = "abc"
> 
> **输出:** [0,6]
> 
> **解释:**
> 
> 起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
> 
> 起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。

**示例 2:**

> **输入:** s = "abab", p = "ab"
> 
> **输出:** [0,1,2]
> 
> **解释:**
> 
> 起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
> 
> 起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
> 
> 起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。

**提示:**
- $1 <= s.length, p.length <= 3 * 10^4$
- `s` 和 `p` 仅包含小写字母

---

## 思路

这道题的一个关键是 **字母异位词**，即字符的种类与个数完全相同的词组。

那么可以构建一个长度固定为 `26` 的数组用来统计字符出现的频次。

先统计字符串 `p` 中每个字符出现的次数，记为 `need`。
然后在 `s` 上维护一个长度最多为 `p.size()` 的滑动窗口，记为 `windows`。

遍历 `s` 时：

- 右指针 `right` 不断向右扩展窗口，并加入当前字符；
- 如果窗口长度**超过** `p` 的长度，就移除左端字符，并右移 `left`；
- 当窗口长度等于 `p` 的长度时，比较 `windows` 和 `need`：
  - 如果两者相同，说明当前窗口是 `p` 的一个字母异位词；
  - 此时将 `left` 加入答案。

---

## 题解

```cpp
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        int len = p.size();
        vector<int> ans;
        vector<int> need(26, 0), windows(26, 0);

        for (char c : p) {
            need[c - 'a']++;
        }

        int left = 0;
        for (int right = 0; right < s.size(); right++) {
            windows[s[right] - 'a']++;
            if (right - left + 1 > len) {
                windows[s[left] - 'a']--;
                left++;
            }
            if (right - left + 1 == len && windows == need) {
                ans.push_back(left);
            }
        }

        return ans;
    }
};
```

## 复杂度分析
时间复杂度 O(n)，一个点是，每次 `if (right - left + 1 == len && windows == need)` 都会比较两个数组，但是长度固定，因此看作常数时间。

空间复杂度 O(1)，因为数组的长度为固定的 26。
